# Electronic Load for Fuel Cell Characterization

**Purpose:**  
Design and development of a controllable electronic load with integrated digital control for real-time fuel cell stack testing up to 160 A.

## 🔧 Overview
This project contains the hardware design, firmware, and documentation for a custom electronic load system used to evaluate automotive fuel cell stacks.

## 📁 Contents
- **Hardware/** – schematics, PCB exports, BOM  
- **Firmware/** – ESP32 source code (C/C++), communication (I²C/SPI)  
- **Docs/** – user manual, test data, thesis report  
- **Images/** – photos and graphs of test setup  

## ⚙️ Features
- Analog + digital mixed-signal design using ADC/DAC and PWM  
- ESP32-based control with real-time current regulation  
- Multi-layer PCB ensuring EMC compliance and signal integrity  
- Tested with 160 A stack load under varying conditions  

## 🧠 Tools
- Altium 365, MATLAB/Simulink, Multisim  
- C/C++ (ESP32 microcontroller)  
- Oscilloscope & Logic Analyzer for debugging  

## 👨‍🔬 My Role
End-to-end design and implementation — circuit design, PCB layout, firmware programming, and system documentation.
